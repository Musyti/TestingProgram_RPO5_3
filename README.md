# Тестирующая программа

Простое приложение для создания и прохождения тестов с весовыми коэффициентами ответов. Подходит для викторин, психологических тестов и опросов с накопительной системой баллов.

## 📥 Скачать

Последнюю версию программы можно скачать на странице [Releases](https://github.com/Musyti/TestingProgram_RPO5_3/releases).

Доступны два варианта:
- **install_test.exe** — установщик программы (рекомендуемый способ)
- **TestingProgram.rar** — архив с готовыми файлами программы

## 🚀 Установка и запуск

### Вариант 1: Установщик (рекомендуемый)
1. Скачайте `install_test.exe` со страницы [Releases](https://github.com/Musyti/TestingProgram_RPO5_3/releases) или по [Ссылке](https://github.com/Musyti/TestingProgram_RPO5_3/releases/download/(v1.0)/install_test.exe).
2. Запустите скачанный файл
3. Следуйте инструкциям установщика
4. После завершения установки запустите программу через ярлык на рабочем столе

### Вариант 2: Архив
1. Скачайте `TestingProgram.rar` со страницы [Releases](https://github.com/Musyti/TestingProgram_RPO5_3/releases)
2. Распакуйте содержимое в любую папку
3. Запустите `TestingProgram.exe`

> **Примечание:** При первом запуске программа автоматически создаст файлы `users.json` и `categories.json` в той же папке.

## 🔑 Начало работы

При первом запуске в системе уже есть тестовый администратор:

| Логин | Пароль | Роль |
|-------|--------|------|
| `Admin` | `Admin` | Тестируемый |
| `Admin1` | `Admin` | Администратор |

Вы можете войти под этой учётной записью или зарегистрировать нового пользователя.

## 👥 Роли пользователей

### Администратор
- Создание новых тестов
- Редактирование существующих тестов
- Удаление тестов
- Управление категориями

### Тестируемый
- Просмотр доступных тестов
- Прохождение тестов
- Получение результатов

## 📝 Структура теста

Каждый тест содержит:
- **10 вопросов** — каждый с 4 вариантами ответа
- **Баллы за ответ** — каждый вариант имеет свой вес
- **3 результата** — диапазоны баллов с названием и описанием

## 📁 Файлы программы

| Файл | Назначение |
|------|------------|
| `TestingProgram.exe` | Исполняемый файл |
| `users.json` | Хранилище пользователей (создаётся автоматически) |
| `categories.json` | Хранилище тестов (создаётся автоматически) |
